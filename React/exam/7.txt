
  Метеостанція

  - Розробити додаток для відображення погоди.
  - Використання: https://open-meteo.com/en/docs/ 
    для отримання інформації про погоду.
  - Додати вибір для вибору інформації про погоду відносно координат.
  - Показати місце в координатах відносно API в гугл картах.

Технічні вимоги:

- Додаток повинен складатись мінімум із 4 сторінок, з адаптивним дизайном,
  та повноцінною версткою кожної сторінки.
- Використовувати React версії 18+.
- Використовувати роутінг - react-router-dom.
- Використовувати state manager redux (краще із redux-toolkit).
- Використовувати функціональні компоненти та хуки.
- Можна використовувати сторонні бібліотеки для стилізаці, спрощення роботи:
  react-bootstrap, mui, formik, yup, axios, propTypes, uiid.
